# NuNuBar 0.12.0

NuNuBar 0.12.0 turns the Air65 V3 keyboard drawing into a real mapping editor.
Select a standard key, choose an action, then create, update, or delete the
device-scoped mapping from the App.

## Key mapping

- Existing verified yellow-key `F24` to Fn/Globe rules are imported as-is.
- New mappings support Fn/Globe, Escape, Return, Backspace, Page Up/Down,
  Home/End, playback, mute, and volume actions.
- A green dot identifies mapped physical keys.
- Knob and native Fn controls remain unavailable until NuNuBar has a verified
  input event for them.
- Every write names the exact Karabiner config and timestamped backup first.
- Unrelated profiles, rules, and file permissions remain unchanged.

The editor replaces the selected key's original action. The technical input
event is visible beside an existing mapping but is no longer presented as the
product workflow.

## Compatibility

- Air65 V3 status lighting remains on the verified official wired interface.
- Air96 V2 ANSI retains its hardware-verified v7 route.
- Air60 V2 ANSI, Air75 V2 ANSI, and Halo75 V2 ANSI remain testing firmware.
- No keyboard firmware or Codex Hook configuration is changed by key mapping.

## 中文说明

NuNuBar 0.12.0 把 Air65 V3 键盘图升级为真正的映射编辑器。点选普通按键、选择
目标动作后，可以直接在 App 中创建、更新或删除设备限定的映射。

- 已验证的黄色键 `F24` 到 Fn/地球键规则会被原样识别，不要求迁移；
- 新映射支持 Fn/地球键、Esc、回车、退格、翻页、Home/End、播放、静音和音量；
- 绿色圆点表示该实体键已经存在映射；
- 旋钮和原生 Fn 暂不开放，直到取得可验证的输入事件；
- 每次写入前仍展示准确配置路径和时间戳备份；
- 不修改其他 Karabiner profile、规则、文件权限、Codex Hooks 或键盘固件。

映射会替代所选键原有功能。底层输入事件只作为当前映射旁边的技术状态显示，
不再作为整个页面的专用说明。
