# NuNuBar 0.13.0

NuNuBar 0.13.0 makes the Air65 V3 knob configurable and adds Codex-focused
actions to the keyboard mapping editor.

## Changes

- The knob exposes three independent inputs: NuPhyIO `F21` left rotation,
  `F22` press, and `F23` right rotation.
- System and Codex actions are shown in separate menu sections.
- Codex actions cover new task, task navigation, panels, terminal, browser,
  keyboard shortcuts, and settings.
- Codex mappings are active only while Codex is frontmost. Other applications
  receive the original key event.
- Every mapping change still shows its exact Karabiner destination and backup
  before writing.

## Compatibility

- Air65 V3 lighting continues to use the physically verified official wired
  HID route without a firmware flash.
- Existing yellow-key `F24` to Fn/Globe and other 0.12.0 mappings are imported.
- Air60 V2, Air75 V2, Air96 V2, and Halo75 V2 remain available with their
  existing firmware and validation status.

NuNuBar 0.13.0 为 Air65 V3 旋钮增加三个独立映射：左旋 `F21`、按下 `F22`、

的安装、固件及验证路线均未改变。
