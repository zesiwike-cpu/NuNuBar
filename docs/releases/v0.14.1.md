# NuNuBar 0.14.1

## Air75 V3

- Added the exact Air75 V3 `19F5:1028` official wired control route.
- Matched NuPhyIO's acknowledged 64-byte `D6` side-light writes: one contiguous
  eight-byte state at offset 9 followed by brightness at offset 10.
- Added official firmware detection and a minimum Air75 V3 requirement of
  `1.0.14.6`; older versions are directed to the separately approved NuPhyIO
  update flow.
- Physically verified a real installed Codex Hook transition from configured
  orange working breathe to green complete solid on Air75 V3.
- Kept Air65 V3 lighting and key mappings, Air96 V2 v7, and all contributor V2
  assets unchanged.

## Air75 V3

- 新增准确 Air75 V3 `19F5:1028` 官方有线控制路线；
- 按 NuPhyIO 实机通信实现带 ACK 的 64 字节 `D6` 侧灯写入：offset 9 连续写入
  8 字节状态，再在 offset 10 写入亮度；
- 增加官方固件检测，Air75 V3 最低要求 `1.0.14.6`；旧版本会引导至需单独确认的
  NuPhyIO 官方升级流程；
- 已在 Air75 V3 上实机确认：系统安装的真实 Codex Hook 会从橙色呼吸工作中切换
  到绿色常亮完成；
- 保留 Air65 V3 灯光与快捷键、Air96 V2 v7，以及全部 V2 贡献者资产。
