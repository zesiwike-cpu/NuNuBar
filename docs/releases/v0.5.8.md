# NuphyBar 0.5.8

First public release. NuphyBar shows local AI agent status on the right side light of a NuPhy Air60 V2.

## Downloads

- `NuphyBar-0.5.8-macOS-arm64.dmg` — macOS 14+ on Apple Silicon
- `NuphyBar-Air60-V2-stable-v7.bin` — **NuPhy Air60 V2 ANSI only**
- `SHA256SUMS` — checksums for the release assets

## Highlights

- Integrates with Codex, Claude Code, OpenCode, Grok Build, Hermes, and OpenClaw.
- Shows a blue wave while working, an amber double pulse while waiting or on error, and a green breathing effect when complete.
- Restores the stock rainbow/battery effect at idle and leaves the left Caps Lock indicator unchanged.
- Sends one BLE HID report only when the state changes; the keyboard renders animations locally.
- Uses a four-byte call-site patch on NuPhy's official Air60 V2 v2.1.5 firmware, with a reproducible builder and layout verification.

## Installation notes

The DMG targets macOS 14+ on Apple Silicon. It is ad-hoc signed and has not been notarized with an Apple Developer ID. If macOS blocks the first launch, right-click the app, select **Open**, and allow Input Monitoring in System Settings.

Before flashing, confirm the exact keyboard model, export the VIA layout, keep the official recovery image, and verify SHA-256 checksums. Never flash the Air60 V2 firmware to another Air V2 size.

## 中文说明

首个公开版本。NuphyBar 让 NuPhy Air60 V2 的右侧灯条显示本机 AI Agent 状态。

### 下载

- `NuphyBar-0.5.8-macOS-arm64.dmg` — macOS 14+，Apple Silicon
- `NuphyBar-Air60-V2-stable-v7.bin` — **仅限 NuPhy Air60 V2 ANSI**
- `SHA256SUMS` — Release 文件校验值

### 主要功能

- 支持 Codex、Claude Code、OpenCode、Grok Build、Hermes 和 OpenClaw；
- 工作中显示蓝色波浪、等待/错误显示琥珀色双脉冲、完成显示绿色呼吸；
- 空闲时恢复原厂彩虹/电量效果，左侧 Caps Lock 指示保持原样；
- Mac 只在状态变化时发送一次 BLE HID 报告，动画由键盘本地渲染；
- Air60 V2 固件基于 NuPhy 官方 v2.1.5 的 4 字节最小调用补丁，并提供可复现构建器和布局验证。

### 安装提示

DMG 使用 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次启动如被 macOS 拦截，请右键应用选择“打开”，并在系统设置中允许输入监控。

固件刷写前必须确认型号、备份 VIA、准备官方恢复文件并核对 SHA-256。Air60 V2 固件不能刷入其他 Air V2 尺寸。
